About This Repository

In most cases, machine learning gets limited to the same call, fit, predict loop. Whereas ML is actually much more than that. This repository is a place containing multiple ML projects which involve all the steps starting from data collection to final model deployment.

The Complete ML Pipeline

  1. Data Collection - Gathering datasets from various sources
  2. Data Cleaning - Handling missing values and data quality issues
  3. Data Wrangling - Transforming and restructuring data
  4. Data Analysis - Statistical analysis and insights
  5. Data Visualization - Creating meaningful visual representations
  6. EDA - Exploratory Data Analysis to understand patterns
  7. Outlier Detection - Identifying and handling anomalies
  8. Creating ML Model - Building and training models
  9. Deployment - Making models accessible through web apps

Technologies Used

Python Flask React scikit-learn pandas NumPy Matplotlib Seaborn Plotly Streamlit

Getting Started

Each project has its own README file with detailed instructions on how to recreate the project on your own system. Navigate to the individual project folders to get started.
